## Project Overview

This project implements machine learning algorithms using Scikit-Learn to classify Iris flowers into three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Models Implemented

- Logistic Regression
- Random Forest Classifier
- Support Vector Machine (SVM)

## Dataset

Scikit-Learn Iris Dataset
